# Bias Event Score System

A full-stack web application for managing and tracking scores in bias-related events. Built with JavaScript, featuring a React client and a Node.js server, deployed live on Vercel.

🔗 **Live Demo:** [bias-event-score-system-rosy.vercel.app](https://bias-event-score-system-rosy.vercel.app)

---

## 📌 About the Project

The Bias Event Score System is designed to record, compute, and display scores for events that involve bias evaluation or assessment. It provides a clean interface for users to interact with scoring data and a robust backend to handle data processing and storage.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/kanikarawat22/bias-event-score-system.git
cd bias-event-score-system
```

2. **Install dependencies for the server**

```bash
cd server
npm install
```

3. **Install dependencies for the client**

```bash
cd ../client
npm install
```

### Running Locally

**Start the server:**

```bash
cd server
npm start
```

**Start the client (in a new terminal):**

```bash
cd client
npm start
```
